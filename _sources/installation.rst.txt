Installation
============

From GitHub
-----------
To install the Spin Sampler library, follow these steps (recomended to work on a virtual environment with ``python 3.9``):

1. Clone the repository:

   .. code-block:: bash

       git clone https://github.com/CristopherErazo/devtools_scicomp_project_2025.git <folder_name>

2. Navigate to the folder directory and install the required dependencies:

   .. code-block:: bash

       pip install -r requirements.txt

4. Install the package:

   .. code-block:: bash

       pip install .

Tests
-----

You can run the unit tests just by running:

    .. code-block:: bash

        pytest
